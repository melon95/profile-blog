# 源码学习

## 目录结构

Mobx 的源码主要分为 4 个部分：

1. api：对外暴露的 API 的实现，比如：
   1. `makeObservable`
   2. `makeAutoObservable`
2. core：核心方法和类的实现，比如：
   1. `startBatch`、`endBatch`
   2. `reportObserved`
   3. `createAction`
   4. `ComputedValue`
3. types：关键类的实现，比如：
   1. `ObservableObjectAdministration`
   2. `ObservableValue`
4. utils：常用工具函数的实现，比如：
   1. `isObject`
   2. `isES6Map`
   3. `addHiddenProp`

## 学习路径

首先会从最常用的 API `makeObservable` 开始，然后一步步的深入，去研究状态的响应式是怎么实现的。

然后再分析 _Reaction_ 的 API 实现，研究 `autorun` 、`reaction` 的底层实现。

最后会分析 Mobx 是怎么与 React 集成的，以及集成工具包 `mobx-react-lite` 的内部实现。
