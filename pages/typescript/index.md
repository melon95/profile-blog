# TypeScript

随着 TypeScript 普及度越来越高，使用 TypeScript 开发的项目也是越来越多，TypeScript 也是越来越重要。

因为前几个使用 TypeScript 开发的项目都是临时看一遍文档就直接开发，慢慢的就变成了 AnyScript，代码质量提高的比较有效，然后就有了这个系列的学习。
